# v0.1.8 开发计划

## 1. 版本定位

`v0.1.8` 计划继续围绕失败复盘体验做小步迭代，把当前已有的失败趋势视图再推进到更适合快速筛查重点异常的方向。

## 2. 版本目标

本版本计划交付以下能力：

- 增强失败趋势的筛查或聚焦体验
- 优化失败复盘路径
- 继续补齐文档与测试

## 3. 范围定义

### 3.1 纳入 `v0.1.8`

- 异常复盘相关审计增强
- README 示例补强
- 文档与测试

### 3.2 暂不纳入 `v0.1.8`

- 多 Agent 团队协作
- 审批协议
- 自动任务拆解
- 自动认领任务
- 可视化界面

## 4. 设计重点

### 4.1 核心原则

- 保持小步迭代
- 优先提升失败筛查时的效率
- 不破坏既有工具边界

### 4.2 版本重点

- 失败趋势的聚焦能力
- 失败复盘路径的顺滑度
- 文档与代码一致性

## 5. 任务拆解

### A. 异常复盘

- 评估是否需要更细的趋势筛查、聚焦或下钻能力
- 优化从趋势概览到详细历史的路径

### B. 演示整理

- 增加更贴近失败趋势筛查的示例
- 保持 README 简洁可执行

### C. 测试与发布

- 增加相关测试
- 更新 README
- 更新版本文档

## 6. 验收清单

- [ ] 失败趋势更容易被筛查和聚焦
- [ ] 失败复盘路径更顺滑
- [ ] 测试通过
- [ ] 文档更新

## 7. 风险提示

- 如果趋势筛查能力继续增加，需要避免摘要过重
- 若输出变得过于密集，可能影响快速浏览
- 仍需持续控制范围膨胀
